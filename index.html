啟發

 打羽球這條路，源自於高一那年我鄰居大哥的熱情邀約。他說：「走啦！打羽球很爽的！」結果我一去，發現他只是缺個撿球的。剛開始我連拍子都碰不到，每次揮空他都在旁邊笑到岔氣。但奇妙的是，越打越上癮，特別是有一次終於把他殺球接住，整個世界都放煙火了！現在想想，當初如果鄰居大哥是打高爾夫，我可能現在在撿球車裡飆速吧。

成長

 從一開始被鄰居大哥拉去打羽球，到後來不僅上教練課，還跟各種陌生人打團打，羽球讓我學會了如何在比賽中面對不同的挑戰。一開始我還是只會站在場邊等球來，完全不懂戰術；但後來越打越有自信，甚至會在場上跟陌生隊友搭配配合。每次與不同的人對打，總能學到新的技巧，還能認識各種奇妙的戰術。有時候對手打得太猛，我都以為自己在打乒乓球！羽球不只鍛鍊了體力，還教會我如何和陌生人建立信任，並一起合作爭取勝利。

爬蟲

因為喜歡戴資穎，所以我嘗試透過網頁爬蟲技術來收集她的主要比賽成績，並整理成結構化資料。我使用 BeautifulSoup 分析比賽成績的 HTML 表格，逐一提取年份、賽事名稱、級別、項目與成績，最後匯出成 CSV 檔案，方便後續分析與查詢。

""" Created on Fri May 9 06:33:45 2025 @author: a0989 """ from selenium import webdriver from selenium.webdriver.chrome.service import Service from selenium.webdriver.common.by import By from selenium.webdriver.common.keys import Keys from selenium.webdriver.support.ui import WebDriverWait from selenium.webdriver.support import expected_conditions as EC import time from webdriver_manager.chrome import ChromeDriverManager import pandas as pd def get_tai_tzu_ying_career_stats(): # 瀏覽器設定 options = webdriver.ChromeOptions() options.add_argument("--disable-blink-features=AutomationControlled") driver = webdriver.Chrome( service=Service(ChromeDriverManager().install()), options=options ) try: # 啟動 driver.get("https://zh.wikipedia.org/wiki/Wikipedia:%E9%A6%96%E9%A1%B5") driver.maximize_window() # 搜尋關鍵字 search_box = WebDriverWait(driver, 10).until( EC.presence_of_element_located((By.CSS_SELECTOR, "input.cdx-text-input__input")) ) search_box.send_keys("戴资颖") search_box.send_keys(Keys.ENTER) time.sleep(2) # 滾動到表格位置 element = WebDriverWait(driver, 10).until( EC.presence_of_element_located((By.ID, "主要比賽成績")) ) driver.execute_script("arguments[0].scrollIntoView({behavior: 'smooth', block: 'center'});", element) time.sleep(2) # 定位表格 table = driver.find_elements(By.CLASS_NAME, "wikitable")[1] # 提取數據 data = [] current_year = "" for row in table.find_elements(By.TAG_NAME, "tr")[1:]: # 跳过表头 cells = row.find_elements(By.TAG_NAME, "td") ths = row.find_elements(By.TAG_NAME, "th") # 處理年份 if len(ths) > 0: current_year = ths[0].text # 提取比赛信息 if len(cells) >= 4: # 确保是行 event = cells[0].text level = cells[1].text category = cells[2].text result = cells[3].text data.append([current_year, event, level, category, result]) # 轉為DataFrame df = pd.DataFrame(data, columns=["年份", "赛事", "级别", "项目", "成绩"]) return df except Exception as e: print(f"发生错误: {str(e)}") return None finally: driver.quit() # 使用示例 if __name__ == "__main__": print("正在获取戴资颖比赛数据...") df = get_tai_tzu_ying_career_stats() if df is not None: print("\n最新5场比赛：") print(df.head()) # 保存結果 df.to_csv(r"‪C:\Users\User\Desktop\戴資穎比賽成績.csv", index=False, encoding="utf-8-sig") print("\n已保存到：戴资颖比赛记录.csv") else: print("获取数据失败")
